# Combined Release of gammaSTAR Parser, gammaSTAR Server and gammaSTAR Reconstructions
This set of tools enables the local installation of a docker container that allows MR simulations using MR Zero (https://github.com/MRsources/MRzero-Core) and an automatic image reconstruction based on gammaSTAR raw representations.

## Prerequisites
1. Have [Docker Desktop](https://docs.docker.com/desktop/) or similar Docker variants installed. 

## How to Use
1. Execute "build_gammastar_server.bat"
2. "docker compose up" inside this main directory
